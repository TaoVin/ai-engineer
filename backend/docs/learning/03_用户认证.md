# 步骤3：用户认证、接口鉴权与 RBAC

## 3.1 知识点概述

本章实现完整的认证鉴权体系：用户登录获取 Token（认证）、请求接口时验证身份（鉴权）、基于角色控制接口访问权限（RBAC）。

### 核心概念

- **认证（Authentication）**：验证「你是谁」—— 登录、JWT Token
- **鉴权（Authorization）**：验证「你能做什么」—— 接口权限检查
- **RBAC（Role-Based Access Control）**：基于角色的权限控制 —— 用户 → 角色 → 权限

### 技术栈

| 库 | 用途 |
|---|------|
| `python-jose[cryptography]` | JWT 编解码 |
| `passlib[bcrypt]` | 密码哈希 |
| `python-multipart` | OAuth2 表单解析 |
| `fastapi.security` | OAuth2、Bearer Token 内置支持 |

### 本章你将学到

1. JWT 双令牌机制（Access Token + Refresh Token）
2. 密码哈希与验证
3. FastAPI 依赖注入实现接口鉴权
4. RBAC 模型设计（五表结构）
5. 自定义装饰器实现权限校验

## 3.2 认证鉴权架构

### 整体流程

```
┌──────────┐     1. 登录（用户名+密码）      ┌──────────────┐
│  客户端   │ ──────────────────────────────→ │  认证服务     │
│          │ ←────────────────────────────── │              │
│          │     2. 返回 Access+Refresh Token │  /auth/login │
└────┬─────┘                                 └──────────────┘
     │
     │  3. 携带 Access Token 请求接口
     │     Authorization: Bearer <token>
     ▼
┌──────────────┐   4. 验证 Token    ┌──────────────┐
│  接口（路由）  │ ────────────────→ │  鉴权中间件   │
│              │                    │              │
│  需要权限：   │   5. 检查权限      │  从 Token 解析│
│  user:read   │ ←──────────────── │  用户 → 角色  │
│              │                    │  → 权限列表   │
│  6. 执行业务  │                    └──────────────┘
└──────────────┘
```

### 双令牌机制

```
Access Token（访问令牌）          Refresh Token（刷新令牌）
┌─────────────────────┐         ┌─────────────────────┐
│ 有效期：15~30 分钟    │         │ 有效期：7 天          │
│ 用途：请求接口携带     │         │ 用途：刷新 Access Token│
│ 存储：前端内存/变量    │         │ 存储：HttpOnly Cookie │
│ 特点：短命、高频使用   │         │ 特点：长命、低频使用   │
└─────────────────────┘         └─────────────────────┘

刷新流程：
Access Token 过期 → 携带 Refresh Token 请求 /auth/refresh
                 → 服务端验证 Refresh Token 有效性
                 → 签发新的 Access Token（+ 新的 Refresh Token）
```

## 3.3 RBAC 模型设计

### 什么是 RBAC

RBAC（Role-Based Access Control）是一种经典的权限管理模型，核心思想是：**用户不直接关联权限，而是通过角色间接关联**。

```
用户（User）──→ 角色（Role）──→ 权限（Permission）
   │                │                │
   │  多对多         │  多对多         │
   ▼                ▼                ▼
 sys_user_role   sys_role_permission
```

### 五表结构

```
┌──────────┐     ┌──────────────┐     ┌──────────┐
│ sys_user │     │ sys_user_role│     │ sys_role │
│          │ ←── │ user_id      │ ──→ │          │
│ id       │     │ role_id      │     │ id       │
│ user_name│     └──────────────┘     │ name     │
│ password │                           │ code     │
└──────────┘                           └────┬─────┘
                                            │
                                    ┌───────────────────┐
                                    │ sys_role_permission│
                                    │ role_id            │
                                    │ permission_id      │
                                    └─────────┬─────────┘
                                              │
                                        ┌─────▼──────────┐
                                        │ sys_permission  │
                                        │                 │
                                        │ id              │
                                        │ name            │
                                        │ code            │
                                        │ type (menu/按钮) │
                                        │ parent_id       │
                                        └─────────────────┘
```

### 权限标识

权限用字符串标识，格式为 `资源:操作`：

```python
# 用户模块
"user:create"    # 创建用户
"user:read"      # 查看用户
"user:update"    # 更新用户
"user:delete"    # 删除用户

# 角色模块
"role:create"
"role:read"
"role:update"
"role:delete"

# 系统管理（管理员专用）
"system:config"  # 系统配置
"system:log"     # 查看日志
```

### RBAC 鉴权流程

```
请求：GET /api/v1/users/
  │
  ▼
1. 从 Token 解析出 user_id = 1
  │
  ▼
2. 查询 user_id=1 关联的角色：[管理员]
  │
  ▼
3. 查询角色「管理员」的权限：[user:create, user:read, user:update, ...]
  │
  ▼
4. 检查接口要求的权限 "user:read" 是否在用户权限列表中
  │
  ├─ 是 → 放行，执行业务逻辑
  └─ 否 → 返回 403 Forbidden
```

## 3.5 安全模块

### 密码哈希与 JWT

**文件位置**：`app/core/security.py`

```python
from datetime import datetime, timedelta, timezone
from typing import Any

from jose import jwt, JWTError
from passlib.context import CryptContext
from app.config.settings import setting

# 密码哈希上下文
pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

# JWT 配置
ALGORITHM = "HS256"


def create_access_token(
    subject: str | int,
    expires_delta: timedelta | None = None,
    extra_data: dict[str, Any] | None = None,
) -> str:
    """
    创建访问令牌

    Args:
        subject: 令牌主体（用户 ID）
        expires_delta: 自定义过期时间
        extra_data: 额外载荷数据

    Returns:
        JWT 令牌字符串
    """
    if expires_delta:
        expire = datetime.now(timezone.utc) + expires_delta
    else:
        expire = datetime.now(timezone.utc) + timedelta(
            minutes=setting.ACCESS_TOKEN_EXPIRE_MINUTES
        )

    to_encode = {"exp": expire, "sub": str(subject), "type": "access"}
    if extra_data:
        to_encode.update(extra_data)

    return jwt.encode(to_encode, setting.SECRET_KEY, algorithm=ALGORITHM)


def create_refresh_token(
    subject: str | int,
    expires_delta: timedelta | None = None,
) -> str:
    """创建刷新令牌"""
    if expires_delta:
        expire = datetime.now(timezone.utc) + expires_delta
    else:
        expire = datetime.now(timezone.utc) + timedelta(
            days=setting.REFRESH_TOKEN_EXPIRE_DAYS
        )

    to_encode = {"exp": expire, "sub": str(subject), "type": "refresh"}
    return jwt.encode(to_encode, setting.SECRET_KEY, algorithm=ALGORITHM)


def verify_token(token: str) -> dict[str, Any] | None:
    """
    验证令牌

    Args:
        token: JWT 令牌

    Returns:
        令牌载荷字典，验证失败返回 None
    """
    try:
        payload = jwt.decode(
            token, setting.SECRET_KEY, algorithms=[ALGORITHM]
        )
        return payload
    except JWTError:
        return None


def verify_password(plain_password: str, hashed_password: str) -> bool:
    """验证密码"""
    return pwd_context.verify(plain_password, hashed_password)


def get_password_hash(password: str) -> str:
    """获取密码哈希"""
    return pwd_context.hash(password)
```

## 3.6 认证依赖（核心）

### 依赖注入实现鉴权

**文件位置**：`app/core/deps.py`

FastAPI 的依赖注入是实现鉴权的核心机制。每个需要鉴权的接口声明一个依赖，FastAPI 自动执行验证逻辑。

```python
from typing import Annotated, AsyncGenerator

from fastapi import Depends, HTTPException, status
from fastapi.security import OAuth2PasswordBearer
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy.orm import selectinload

from app.core.database import async_session_factory
from app.core.security import verify_token
from app.models.user import User

# OAuth2 方案：告诉 Swagger 登录地址
oauth2_scheme = OAuth2PasswordBearer(tokenUrl="/api/v1/auth/login")


async def get_db() -> AsyncGenerator[AsyncSession, None]:
    """获取数据库会话"""
    async with async_session_factory() as session:
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise


# ============================================================
# 第一层：解析 Token → 获取用户
# ============================================================

async def get_current_user(
    token: Annotated[str, Depends(oauth2_scheme)],
    db: Annotated[AsyncSession, Depends(get_db)],
) -> User:
    """
    从请求头解析 Token，获取当前用户

    流程：
    1. 从 Authorization: Bearer <token> 提取 token
    2. 解码 JWT，获取 user_id
    3. 查询数据库获取用户（同时加载角色和权限）
    """
    credentials_exception = HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="无法验证凭据",
        headers={"WWW-Authenticate": "Bearer"},
    )

    # 解码 Token
    payload = verify_token(token)
    if payload is None:
        raise credentials_exception

    # 验证令牌类型（必须是 access token）
    if payload.get("type") != "access":
        raise credentials_exception

    # 获取用户 ID
    user_id: str | None = payload.get("sub")
    if user_id is None:
        raise credentials_exception

    # 查询用户（预加载角色和权限）
    result = await db.execute(
        select(User)
        .options(selectinload(User.roles))
        .where(User.id == int(user_id))
    )
    user = result.scalar_one_or_none()

    if user is None:
        raise credentials_exception

    if not user.is_active:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST, detail="用户未激活"
        )

    return user


# ============================================================
# 第二层：基础鉴权（是否已登录）
# ============================================================

async def get_current_active_user(
    current_user: Annotated[User, Depends(get_current_user)],
) -> User:
    """获取当前已激活用户"""
    if not current_user.is_active:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST, detail="用户未激活"
        )
    return current_user


# ============================================================
# 第三层：管理员鉴权
# ============================================================

async def get_current_superuser(
    current_user: Annotated[User, Depends(get_current_user)],
) -> User:
    """获取当前超级管理员"""
    if not current_user.is_superuser:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN, detail="权限不足，需要管理员权限"
        )
    return current_user
```

### 依赖层级关系

```
oauth2_scheme（提取 Token 字符串）
       │
       ▼
get_current_user（Token → 用户对象，含角色）
       │
       ├──→ get_current_active_user（检查 is_active）
       │
       ├──→ get_current_superuser（检查 is_superuser）
       │
       └──→ require_permission("user:read")（检查 RBAC 权限）
```

## 3.7 RBAC 权限检查

### 权限检查依赖工厂

**文件位置**：`app/core/permissions.py`

```python
from typing import Annotated
from fastapi import Depends, HTTPException, status
from app.core.deps import get_current_user
from app.models.user import User


def get_user_permission_codes(user: User) -> set[str]:
    """
    获取用户所有权限标识集合

    Args:
        user: 用户对象（已预加载 roles）

    Returns:
        权限标识集合，如 {"user:read", "user:create", ...}
    """
    # 超级管理员拥有所有权限
    if user.is_superuser:
        return {"*"}

    permissions = set()
    for role in user.roles:
        if role.is_active:
            for perm in role.permissions:
                permissions.add(perm.code)
    return permissions


class require_permission:
    """
    权限检查依赖类（可调用类）

    用法：
        @router.get("/users", dependencies=[Depends(require_permission("user:read"))])
        async def list_users():
            ...

    或者：
        async def list_users(
            user: Annotated[User, Depends(require_permission("user:read"))]
        ):
            ...
    """

    def __init__(self, permission_code: str):
        self.permission_code = permission_code

    async def __call__(
        self,
        current_user: Annotated[User, Depends(get_current_user)],
    ) -> User:
        """检查当前用户是否拥有指定权限"""
        user_permissions = get_user_permission_codes(current_user)

        # 超级管理员拥有所有权限
        if "*" in user_permissions:
            return current_user

        if self.permission_code not in user_permissions:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail=f"权限不足，需要权限：{self.permission_code}",
            )

        return current_user


class require_any_permission:
    """
    检查用户是否拥有任意一个权限

    用法：
        @router.get("/users", dependencies=[
            Depends(require_any_permission("user:read", "user:manage"))
        ])
    """

    def __init__(self, *permission_codes: str):
        self.permission_codes = set(permission_codes)

    async def __call__(
        self,
        current_user: Annotated[User, Depends(get_current_user)],
    ) -> User:
        user_permissions = get_user_permission_codes(current_user)

        if "*" in user_permissions:
            return current_user

        if not self.permission_codes & user_permissions:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail=f"权限不足，需要以下权限之一：{', '.join(self.permission_codes)}",
            )

        return current_user
```

## 3.9 认证接口

### Auth 路由

**文件位置**：`app/api/v1/endpoints/auth.py`

```python
from typing import Annotated

from fastapi import APIRouter, Depends, HTTPException, status
from fastapi.security import OAuth2PasswordRequestForm
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from app.core.deps import get_db, get_current_user
from app.core.security import (
    verify_password,
    get_password_hash,
    create_access_token,
    create_refresh_token,
    verify_token,
)
from app.models.user import User
from app.schemas.token import Token, RefreshTokenRequest
from app.schemas.user import UserCreate, UserResponse, UserLogin
from app.schemas.base import ResponseBase

router = APIRouter()


@router.post("/register", response_model=ResponseBase[UserResponse])
async def register(
    user_in: UserCreate,
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """用户注册"""
    # 检查用户名是否已存在
    result = await db.execute(
        select(User).where(User.user_name == user_in.user_name)
    )
    if result.scalar_one_or_none():
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST, detail="用户名已存在"
        )

    # 创建用户
    user = User(
        user_name=user_in.user_name,
        nick_name=user_in.nick_name,
        email=user_in.email,
        password=get_password_hash(user_in.password),
    )
    db.add(user)
    await db.flush()
    await db.refresh(user)

    return ResponseBase(data=UserResponse.model_validate(user))


@router.post("/login", response_model=ResponseBase[Token])
async def login(
    user_in: UserLogin,
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """
    用户登录（JSON 模式）

    返回 Access Token + Refresh Token
    """
    # 查询用户
    result = await db.execute(
        select(User).where(User.user_name == user_in.user_name)
    )
    user = result.scalar_one_or_none()

    if not user or not verify_password(user_in.password, user.password):
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="用户名或密码错误"
        )

    if not user.is_active:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST, detail="用户未激活"
        )

    # 生成双令牌
    access_token = create_access_token(subject=user.id)
    refresh_token = create_refresh_token(subject=user.id)

    return ResponseBase(
        data=Token(
            access_token=access_token,
            refresh_token=refresh_token,
        )
    )


@router.post("/login/form", response_model=ResponseBase[Token])
async def login_form(
    db: Annotated[AsyncSession, Depends(get_db)],
    form_data: Annotated[OAuth2PasswordRequestForm, Depends()],
):
    """
    用户登录（OAuth2 表单模式）

    Swagger 文档中的 Authorize 按钮使用此接口
    """
    result = await db.execute(
        select(User).where(User.user_name == form_data.username)
    )
    user = result.scalar_one_or_none()

    if not user or not verify_password(form_data.password, user.password):
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="用户名或密码错误",
            headers={"WWW-Authenticate": "Bearer"},
        )

    access_token = create_access_token(subject=user.id)
    refresh_token = create_refresh_token(subject=user.id)

    return ResponseBase(
        data=Token(access_token=access_token, refresh_token=refresh_token)
    )


@router.post("/refresh", response_model=ResponseBase[Token])
async def refresh_token(
    token_in: RefreshTokenRequest,
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """刷新 Access Token"""
    payload = verify_token(token_in.refresh_token)
    if not payload or payload.get("type") != "refresh":
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="无效的刷新令牌"
        )

    user_id = payload.get("sub")
    result = await db.execute(select(User).where(User.id == int(user_id)))
    user = result.scalar_one_or_none()

    if not user or not user.is_active:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="用户不存在或未激活"
        )

    new_access_token = create_access_token(subject=user.id)
    new_refresh_token = create_refresh_token(subject=user.id)

    return ResponseBase(
        data=Token(
            access_token=new_access_token, refresh_token=new_refresh_token
        )
    )


@router.get("/me", response_model=ResponseBase[UserResponse])
async def get_me(
    current_user: Annotated[User, Depends(get_current_user)],
):
    """获取当前登录用户信息"""
    return ResponseBase(data=UserResponse.model_validate(current_user))
```

## 3.10 RBAC 管理接口

### CRUD 操作

**文件位置**：`app/crud/crud_role.py`

```python
from typing import Any
from sqlalchemy import select
from sqlalchemy.orm import selectinload
from sqlalchemy.ext.asyncio import AsyncSession

from app.crud.base import CRUDBase
from app.models.role import Role
from app.models.user import User
from app.models.association import sys_user_role


class CRUDRole(CRUDBase[Role]):
    """角色 CRUD"""

    async def get_by_code(self, db: AsyncSession, *, code: str) -> Role | None:
        """根据角色标识获取角色"""
        result = await db.execute(select(Role).where(Role.code == code))
        return result.scalar_one_or_none()

    async def get_with_permissions(
        self, db: AsyncSession, *, id: int
    ) -> Role | None:
        """获取角色（含权限列表）"""
        result = await db.execute(
            select(Role)
            .options(selectinload(Role.permissions))
            .where(Role.id == id)
        )
        return result.scalar_one_or_none()

    async def assign_permissions(
        self, db: AsyncSession, *, role: Role, permission_ids: list[int]
    ) -> Role:
        """分配权限给角色"""
        from app.models.permission import Permission

        result = await db.execute(
            select(Permission).where(Permission.id.in_(permission_ids))
        )
        permissions = list(result.scalars().all())
        role.permissions = permissions
        await db.flush()
        await db.refresh(role)
        return role


class CRUDUser(CRUDBase[User]):
    """用户 CRUD（扩展）"""

    async def get_by_username(
        self, db: AsyncSession, *, username: str
    ) -> User | None:
        result = await db.execute(
            select(User).where(User.user_name == username)
        )
        return result.scalar_one_or_none()

    async def get_with_roles(
        self, db: AsyncSession, *, id: int
    ) -> User | None:
        """获取用户（含角色列表）"""
        result = await db.execute(
            select(User)
            .options(selectinload(User.roles))
            .where(User.id == id)
        )
        return result.scalar_one_or_none()

    async def assign_roles(
        self, db: AsyncSession, *, user: User, role_ids: list[int]
    ) -> User:
        """分配角色给用户"""
        result = await db.execute(
            select(Role).where(Role.id.in_(role_ids))
        )
        roles = list(result.scalars().all())
        user.roles = roles
        await db.flush()
        await db.refresh(user)
        return user


role_crud = CRUDRole(Role)
user_crud = CRUDUser(User)
```

```python
from typing import Annotated

from fastapi import APIRouter, Depends, HTTPException, status
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from app.core.deps import get_db, get_current_user
from app.core.permissions import require_permission
from app.models.user import User
from app.models.role import Role
from app.crud.crud_role import role_crud
from app.schemas.role import RoleCreate, RoleUpdate, RoleResponse
from app.schemas.base import ResponseBase

router = APIRouter()


@router.get(
    "/",
    response_model=ResponseBase[list[RoleResponse]],
    dependencies=[Depends(require_permission("role:read"))],
)
async def list_roles(
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """获取角色列表（需要 role:read 权限）"""
    roles = await role_crud.get_multi(db)
    return ResponseBase(
        data=[RoleResponse.model_validate(r) for r in roles]
    )


@router.post(
    "/",
    response_model=ResponseBase[RoleResponse],
    dependencies=[Depends(require_permission("role:create"))],
)
async def create_role(
    role_in: RoleCreate,
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """创建角色（需要 role:create 权限）"""
    existing = await role_crud.get_by_code(db, code=role_in.code)
    if existing:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST, detail="角色标识已存在"
        )

    role = await role_crud.create(db, obj_in=role_in.model_dump())
    return ResponseBase(data=RoleResponse.model_validate(role))


@router.put(
    "/{role_id}",
    response_model=ResponseBase[RoleResponse],
    dependencies=[Depends(require_permission("role:update"))],
)
async def update_role(
    role_id: int,
    role_in: RoleUpdate,
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """更新角色（需要 role:update 权限）"""
    role = await role_crud.get(db, id=role_id)
    if not role:
        raise HTTPException(status_code=404, detail="角色不存在")

    update_data = role_in.model_dump(exclude_unset=True)
    role = await role_crud.update(db, db_obj=role, obj_in=update_data)
    return ResponseBase(data=RoleResponse.model_validate(role))


@router.delete(
    "/{role_id}",
    dependencies=[Depends(require_permission("role:delete"))],
)
async def delete_role(
    role_id: int,
    db: Annotated[AsyncSession, Depends(get_db)],
):
    """删除角色（需要 role:delete 权限）"""
    role = await role_crud.remove(db, id=role_id)
    if not role:
        raise HTTPException(status_code=404, detail="角色不存在")
    return ResponseBase(msg="删除成功")
```

### 方式一：仅需登录

```python
@router.get("/me")
```

## 3.14 常见问题

### Q1: Token 过期后怎么办？

客户端检测到 401 响应后，携带 Refresh Token 请求 `/auth/refresh` 获取新的 Access Token。

### Q2: 超级管理员为什么不需要查权限表？

`is_superuser=True` 的用户在 `get_user_permission_codes` 中直接返回 `{"*"}`，跳过权限表查询，简化逻辑。

### Q3: 如何在接口中同时获取用户和检查权限？

```python
@router.get("/users")
async def list_users(
    current_user: Annotated[User, Depends(require_permission("user:read"))],
):
    # current_user 既是当前用户，也已通过权限检查
    ...
```

### Q4: 如何给接口添加多个权限检查？

```python
@router.get(
    "/admin/dashboard",
    dependencies=[
        Depends(require_permission("dashboard:view")),
        Depends(require_permission("admin:access")),
    ],
)
```
